# Food Images

This directory contains placeholder images for the menu items. Currently, all images are SVG placeholders.

## Replacing Images

To replace with actual food photos:

1. **Option 1: Use Your Own Photos**
   - Take high-quality photos of your dishes (800x600px minimum)
   - Ensure good lighting and composition
   - Save as JPG format
   - Replace the corresponding file in this directory

2. **Option 2: Use Stock Photos**
   - Sites like Unsplash, Pexels, or Pixabay offer free high-quality food photos
   - Search for "indian food", specific dish names, etc.
   - Download and save with the exact filename listed below
   - Ensure you comply with the license terms

3. **Option 3: Professional Photography**
   - Hire a food photographer
   - Professional photos greatly improve conversion rates
   - Investment typically pays for itself in increased orders

## Image Specifications

- **Dimensions**: 800x600px minimum (1200x900px recommended)
- **Format**: JPG (optimized for web)
- **File Size**: Keep under 200KB per image
- **Aspect Ratio**: 4:3 preferred
- **Quality**: 80-85% JPEG quality

## Optimization

Before uploading, optimize images using:
- [TinyJPG](https://tinyjpg.com/) - Free image compression
- [Squoosh](https://squoosh.app/) - Advanced compression tool
- `npm run optimize-images` - If you add an optimization script
